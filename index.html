What is it ?

Sensorendipity is an open source android based toolkit, giving users the real-time understanding of sensor values which can be further used: Making sense of the Data. The data generated by sensors is in the form of JSON string, allowing you to develop anything you want using one or more sensor values. The curated data can be used to detect a particular motion or gesture.
